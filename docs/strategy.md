# FlowKits.ai Strategy — Updated April 15, 2026
## CEO Review + Will's Direction

### The Core Bet
AI agent workflows are the next Notion templates / Etsy digital products. Founders want battle-tested workflows, not another weekend of prompt engineering. Build once, sell many times. Zero compute. Sell the recipe, not the kitchen.

### The Funnel
1. **n8ntocode.com** (existing, live) — Convert n8n workflows to Python. Free first conversion, $5 after. Top-of-funnel.
2. **FlowKits.ai** (marketplace, build next) — Premium verified AI agent workflows. The upgrade path from n8n.

### What We Own
- n8ntocode.com — live product. Converts n8n JSON → Python + AGENTS.md. 80+ node types. $5/conversion.
- FlowKits.ai — domain (confirm if anything live)
- @FlowKitsAI on X — premium account, 3 tweets/day pipeline working
- Expert Tweeter kit on JourneyKits.ai
- VibeCodingDad.com — Chrome extensions, ebook, existing products

### Competitive Intel
- **ClawMart** (shopclawmart.com): 2,761 listings, $100K+ paid out, 90% creator rev share. Quality is garbage, most have 0-2 reviews. OpenClaw-centric.
- **flowkit.in**: 46 free n8n workflows, 97 GitHub stars, open source, no monetization. Similar name, different domain (.in vs .ai). Proves demand for organized n8n workflows.
- **n8n**: 9,167 free templates. Deterministic, brittle, no AI reasoning. The pain point we solve.
- **GPT Store**: 3M+ GPTs. Terrible discovery, no quality control. OpenAI owns distribution.

### What We Sell
NOT a marketplace yet. A storefront. Our own verified workflows first.
- Source: n8n top workflows, converted via n8ntocode.com
- Verification: each converted workflow tested end-to-end
- Credit original n8n authors, recruit them as marketplace sellers later
- Free tier: basic converted workflows (lead gen, email, scheduling)
- Premium tier: enhanced with AI reasoning, error handling, autonomous operation ($15-50)

### Revenue Model
- **Open-source all kit code** (free download, read every line, run yourself) — code is marketing
- **Paid monitoring tier** (execution dashboard, failure alerts, auto-updates) — this is the product
- Marketplace fee (10-15% when we open to creators, matching ClawMart's 10%)
- Subscription curation ($9-15/mo for premium access) — later
- Observability/monitoring tier (execution dashboard, failure alerts) — at launch, drives subscription conversion
- n8ntocode.com ($5/conversion) — separate revenue stream, feeds FlowKits funnel
- Model: like Terraform (free modules, paid cloud). Piracy = distribution.

### Revenue Target
$100K. Path: 50 workflows x $25 avg x 20 sales = $25,000 from own products. Marketplace + subs for the rest.

### The n8n Conversion Thesis
- n8ntocode already converts n8n JSON → Python + AGENTS.md
- Conversion is one-time, automated, near-zero cost
- n8n workflows are largely MIT-licensed (transformative use, credit original)
- NOT "unlimited workflows" dumping — verify each one, quality is the brand
- Use conversion as marketing content (tweet the process), not just inventory

### Key Risks
1. **Timing** — agent marketplace may become a platform feature (OpenAI, Anthropic)
2. **Quality on conversion** — converted code must actually work
3. **Brand confusion** — flowkit.in exists (46 workflows, .in domain)
4. **Free competition** — n8n templates are free, GPT Store is free
5. **Hits-driven business** — top 5% make 95% of revenue (like app stores)

### Mitigants
1. Build fast, capture the window. n8ntocode as funnel buys time.
2. Verify every workflow. "Every workflow on FlowKits is tested" as brand promise.
3. FlowKits.ai is .ai domain, different positioning (AI-native vs n8n templates)
4. Sell verification, curation, and AI-native upgrade. Not the workflow itself.
5. Start with our own high-quality workflows, not a firehose of converted junk.
